    makeKey(arrival):
    - Input: single arrival object
    - Returns: string "${tripId}_${serviceDate}"
    - Both fields are required (function assumes valid data)

    diffArrivals(prevArrivals, currentArrivals, now):
    - Filters departed from currentArrivals using filterDeparted

    - If prevArrivals is null or empty, all current arrivals are _isNew = true
    - Returns the filtered+tagged array (does NOT mutate input)
    - Preserves original order of currentArrivals
